# WildBerryEye
## Backend
- Create your venv with system packages enabled:
$ python3 -m venv --system-site-packages venv
- Run Flask App
$ python3 app.py

- Activate it:
$ source venv/bin/activate

- Install Pacakes
$ pip install -r requirements.txt

## Frontend
- Install all Packages
$ npm install

- Run
$ npm start
